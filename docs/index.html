Como Instalar e Usar

1

Instale o Motor Ghostscript (Obrigatório)

O compressor não funcionará na sua taxa máxima sem este motor matemático gratuito instalado passivamente no seu sistema Windows.

Baixar o Ghostscript Core
2

Baixe o Executável (.exe)

Baixe no botão acima a versão pré-compilada, dispensando a instalação de Python ou dependências e abra imediatamente (PDF Compressor.exe).

3

Selecione seu modo de segurança

Modo Leve / Médio: Ideal para reduzir PDFs padronizados com poucas imagens, mantendo resolução total.

Modo Extremo: Acionado com algoritmo `Subsample` 85-DPI agressivo feito perfeitamente para arquivos gigas provenientes de HTML2Canvas / JPEGs massivos.
